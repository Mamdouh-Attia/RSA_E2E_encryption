# How to run application

## For normal chatting

_Just run the server.py at a terminal and the client.py at another terminal_


• Server:
```bash
Python ./server.py
```
• Client:
```bash
Python ./client.py
```
_now you can start chatting together!_

## For breaking program

_Just run the breaker.py on terminal_

```bash
Python ./breaker.py
```
